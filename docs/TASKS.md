## 技术债

1. **Right Shift 兼容性** - 特殊键盘下可能仍需继续打磨
2. **高权限窗口注入** - 普通权限程序无法向管理员窗口注入
3. **IME 兼容性** - 第一版使用剪贴板方案，部分 IME 场景可能有问题
4. **目标窗口锁定** - 当前仍有继续优化空间
5. **图标设计** - 使用默认图标，后续需要设计专用图标
