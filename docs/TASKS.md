# 语音输入法 - 执行清单

## 当前阶段目标

完成 Windows 11 桌面常驻语音输入助手 MVP，并持续优化实时输入体验：
- 托盘常驻 + Right Shift 激活
- 录音 + 波形显示
- 云端流式 ASR 识别
- 文本注入
- 可选 LLM 纠错
- 热键稳定性、注入可靠性、并发安全性全面优化

## 技术债（剩余）

1. **高权限窗口注入** - 普通权限程序无法向管理员窗口注入，可考虑提示用户手动粘贴
2. **IME 兼容性** - 剪贴板方案在部分 IME 场景下可能有问题
3. **图标设计** - 使用默认图标，后续需要设计专用图标
4. **Empty or null text 日志** - 火山 ASR 持续返回空响应已降噪为 Debug 级别

---

## 已解决的技术债

1. ~~**Right Shift 兼容性**~~ - 已通过业务层自动停止兜底解决，不再死磕键盘钩子
2. ~~**热键释放卡界面**~~ - SemaphoreSlim + 自动停止双条件兜底
3. ~~**粘滞键/筛选键弹出**~~ - 全面禁用辅助功能热键
4. ~~**音频服务 double dispose**~~ - Interlocked.Exchange 统一释放
5. ~~**WebSocket 连接泄漏**~~ - 超时 + 先关后放
6. ~~**HttpClient 连接池耗尽**~~ - 静态单例复用
7. ~~**设置文件损坏**~~ - 原子写入 + 备份
8. ~~**修饰键误伤**~~ - 只释放实际使用的 Ctrl
9. ~~**剪贴板注入过于严格**~~ - 放宽前景窗口校验
